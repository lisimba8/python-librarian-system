This is program that can be used by a librarian to allow members to borrow 
and return books to a libarary. The system also provides functinality to 
recommend books to a user (new or returning), the functionality to search
for a specific book in the database using it's name.
The databse.txt file contains the data on each book. 
The log.txt file contains the logs of all transactions that have occurred in 
the library. 
The main module of this program is menu.py and this is the module that should 
be called in order to start the application.
Notice how the most popular genre graph is embedded in the frame and how a 
user can get suggested list of books even if they have not checkout any books
before.
